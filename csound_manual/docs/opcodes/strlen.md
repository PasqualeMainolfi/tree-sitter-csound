# strlen
Return the length of a string, or zero if it is empty. strlen runs at init time only.

## Syntax
=== "Modern"
    ``` csound-orc
    ilen = strlen(Sstr)
    ```

=== "Classic"
    ``` csound-orc
    ilen strlen Sstr
    ```
