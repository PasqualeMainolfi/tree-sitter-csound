# butbp
Same as the [butterbp](../opcodes/butterbp.md) opcode.

## Syntax
=== "Modern"
    ``` csound-orc
    ares = butbp(asig, kfreq, kband [, iskip])
    ```

=== "Classic"
    ``` csound-orc
    ares butbp asig, kfreq, kband [, iskip]
    ```
