# butbr
Same as the [butterbr](../opcodes/butterbr.md) opcode.

## Syntax
=== "Modern"
    ``` csound-orc
    ares = butbr(asig, kfreq, kband [, iskip])
    ```

=== "Classic"
    ``` csound-orc
    ares butbr asig, kfreq, kband [, iskip]
    ```
