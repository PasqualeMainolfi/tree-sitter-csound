# strlowerk
Convert Ssrc to lower case, and write the result to Sdst.

_strlowerk_ runs both at init and performance time.

## Syntax
=== "Modern"
    ``` csound-orc
    Sdst = strlowerk(Ssrc)
    ```

=== "Classic"
    ``` csound-orc
    Sdst strlowerk Ssrc
    ```
