# **Reverberation**

The opcodes one can use for reverberation are:

* [alpass](../opcodes/alpass.md)
* [babo](../opcodes/babo.md)
* [comb](../opcodes/comb.md)
* [combinv](../opcodes/combinv.md)
* [freeverb](../opcodes/freeverb.md)
* [nestedap](../opcodes/nestedap.md)
* [nreverb](../opcodes/nreverb.md) (also called [reverb2](../opcodes/reverb2.md))
* [platerev](../opcodes/platerev.md)
* [reverb](../opcodes/reverb.md)
* [reverbsc](../opcodes/reverbsc.md)
* [valpass](../opcodes/valpass.md)
* [vcomb](../opcodes/vcomb.md)
