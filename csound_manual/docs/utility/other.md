# **Other Csound Utilities (CS, CSB64ENC, DNOISE, ENVEXT, EXTRACTOR, MAKECSD, MIXER, SCALE)**

The following miscellaneous utilities are available:

* [CS](cs.md): Starts Csound with a set of options that can be controlled by environment variables, and input and output files determined by the specified filename stem.
* [CSB64ENC](csb64enc.md): Converts a binary file to a Base64 enconded text file.
* [DNOISE](dnoise.md): Reduces noise in a file.
* [ENVEXT](envext.md): Extract the envelope of a file to a text list.
* [EXTRACTOR](extractor.md): Extract a section of audio from an audio file.
* [MAKECSD](makecsd.md): Creates a CSD file from the specified input files.
* [MIXER](mixer.md): Mixes together a number of soundfiles.
* [SCALE](scale.md): Scale the amplitude of a sound file.
