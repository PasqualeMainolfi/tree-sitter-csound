# **Dynamic Spectrum Oscillators**

The opcodes that generate dynamic spectra are:

- Harmonic spectra: [buzz](../opcodes/buzz.md) and [gbuzz](../opcodes/gbuzz.md)
- Impulse generator: [mpulse](../opcodes/mpulse.md)
- Bandlimited shape-shifting square-pulse-saw-sinewave oscillator: [squinewave](../opcodes/squinewave.md)
- Band limited oscillators (analog modelled): [vco](../opcodes/vco.md) and [vco2](../opcodes/vco2.md)

The following opcodes can be used to generate band-limited waveforms for use with [vco2](../opcodes/vco2.md) and other oscillators:

- [vco2init](../opcodes/vco2init.md)
- [vco2ft](../opcodes/vco2ft.md)
- [vco2ift](../opcodes/vco2ift.md)
