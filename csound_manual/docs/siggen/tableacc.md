# **Table Access**

The opcodes that access tables are:

* [oscil1](../opcodes/oscil1.md)
* [oscil1i](../opcodes/oscil1i.md)
* [osciln](../opcodes/osciln.md)
* [oscilx](../opcodes/oscilx.md)
* [table](../opcodes/table.md)
* [table3](../opcodes/table3.md)
* [tablei](../opcodes/tablei.md)

Opcodes ending in 'i' implement linear interpolation and opcodes ending in '3' implement cubic interpolation.

The following opcodes implement fast table reading/writing without boundary checks:

* [tab](../opcodes/tab.md)
* [tab_i](../opcodes/tab_i.md)
* [tabw](../opcodes/tabw.md)
* [tabw_i](../opcodes/tabw_i.md)

See the sections [Table Queries](../table/queries.md),
[Read/Write Operations](../table/readwrit.md) and
[Table Reading with Dynamic Selection](../table/select.md)
for other table operations.
