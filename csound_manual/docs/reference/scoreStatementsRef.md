# **Score Statements**

[a](../scoregens/a.md) - Advance score time by a specified amount.<br>
[b](../scoregens/b.md) - Resets the clock.<br>
[B](../scoregens/capB.md) - Accumulate clock adjustment.<br>
[C](../scoregens/C.md) - Toggles carry facility.<br>
[d](../scoregens/d.md) - Deletes an infinite instrument.<br>
[e](../scoregens/e.md) - Marks the end of the last section of the score.<br>
[f](../scoregens/f.md) - Causes a [GEN subroutine](../score/genroutines.md) to place values in a stored function table.<br>
[i](../scoregens/i.md) - Makes an instrument active at a specific time and for a certain duration.<br>
[m](../scoregens/m.md) - Sets a named mark in the score.<br>
[n](../scoregens/n.md) - Repeats a section.<br>
[q](../scoregens/q.md) - Used to quiet an instrument.<br>
[r](../scoregens/r.md) - Starts a repeated section.<br>
[s](../scoregens/s.md) - Marks the end of a section.<br>
[t](../scoregens/t.md) - Sets the tempo.<br>
[v](../scoregens/v.md) - Provides for locally variable time warping of score events.<br>
[x](../scoregens/x.md) - Skip the rest of the current section.<br>
[y](../scoregens/y.md) - Set seed for random numbers, either from p1 or, if omitted, the clock.<br>
[{](../scoregens/leftbrace.md) - Begins a non-sectional, nestable loop.<br>
[}](../scoregens/rightbrace.md) - Ends a non-sectional, nestable loop.<br>
