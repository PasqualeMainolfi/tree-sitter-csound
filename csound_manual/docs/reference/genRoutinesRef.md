# **GEN Routines**

[GEN01](../scoregens/gen01.md) - Transfers data from a soundfile into a function table.<br>
[GEN02](../scoregens/gen02.md) - Transfers data from immediate pfields into a function table.<br>
[GEN03](../scoregens/gen03.md) - Generates a stored function table by evaluating a polynomial.<br>
[GEN04](../scoregens/gen04.md) - Generates a normalizing function.<br>
[GEN05](../scoregens/gen05.md) - Constructs functions from segments of exponential curves.<br>
[GEN06](../scoregens/gen06.md) - Generates a function comprised of segments of cubic polynomials.<br>
[GEN07](../scoregens/gen07.md) - Constructs functions from segments of straight lines.<br>
[GEN08](../scoregens/gen08.md) - Generate a piecewise cubic spline curve.<br>
[GEN09](../scoregens/gen09.md) - Composite waveforms made up of weighted sums of simple sinusoids.<br>
[GEN10](../scoregens/gen10.md) - Composite waveforms made up of weighted sums of simple sinusoids.<br>
[GEN11](../scoregens/gen11.md) - Additive set of cosine partials.<br>
[GEN12](../scoregens/gen12.md) - Generates the log of a modified Bessel function of the second kind.<br>
[GEN13](../scoregens/gen13.md) - Stores a polynomial whose coefficients derive from the Chebyshev polynomials of the first kind.<br>
[GEN14](../scoregens/gen14.md) - Stores a polynomial whose coefficients derive from Chebyshevs of the second kind.<br>
[GEN15](../scoregens/gen15.md) - Creates two tables of stored polynomial functions.<br>
[GEN16](../scoregens/gen16.md) - Creates a table from a starting value to an ending value.<br>
[GEN17](../scoregens/gen17.md) - Creates a step function from given x-y pairs.<br>
[GEN18](../scoregens/gen18.md) - Writes composite waveforms made up of pre-existing waveforms.<br>
[GEN19](../scoregens/gen19.md) - Composite waveforms made up of weighted sums of simple sinusoids.<br>
[GEN20](../scoregens/gen20.md) - Generates functions of different windows.<br>
[GEN21](../scoregens/gen21.md) - Generates tables of different random distributions.<br>
[GEN23](../scoregens/gen23.md) - Reads numeric values from a text file.<br>
[GEN24](../scoregens/gen24.md) - Reads numeric values from another allocated function-table and rescales them.<br>
[GEN25](../scoregens/gen25.md) - Construct functions from segments of exponential curves in breakpoint fashion.<br>
[GEN27](../scoregens/gen27.md) - Construct functions from segments of straight lines in breakpoint fashion.<br>
[GEN28](../scoregens/gen28.md) - Reads a text file which contains a time-tagged trajectory.<br>
[GEN30](../scoregens/gen30.md) - Generates harmonic partials by analyzing an existing table.<br>
[GEN31](../scoregens/gen31.md) - Mixes any waveform specified in an existing table.<br>
[GEN32](../scoregens/gen32.md) - Mixes any waveform, resampled with either FFT or linear interpolation.<br>
[GEN33](../scoregens/gen33.md) - Generate composite waveforms by mixing simple sinusoids.<br>
[GEN34](../scoregens/gen34.md) - Generate composite waveforms by mixing simple sinusoids.<br>
[GEN40](../scoregens/gen40.md) - Generates a random distribution using a distribution histogram.<br>
[GEN41](../scoregens/gen41.md) - Generates a random list of numerical pairs.<br>
[GEN42](../scoregens/gen42.md) - Generates a random distribution of discrete ranges of values.<br>
[GEN43](../scoregens/gen43.md) - Loads a PVOCEX file containing a PV analysis.<br>
[GEN44](../scoregens/gen44.md) - Creates a stifness matrix for scan/scanu.<br>
[GEN49](../scoregens/gen49.md) - Transfers data from an MP3 soundfile into a function table.<br>
[GEN51](../scoregens/gen51.md) - fills a table with a fully customized micro-tuning scale, in the manner of Csound opcodes _cpstun_, _cpstuni_ and _cpstmid_.<br>
[GEN52](../scoregens/gen52.md) - Creates an interleaved multichannel table from the specified source tables, in the format expected by the _ftconv_ opcode.<br>
[GEN53](../scoregens/gen53.md) - creates a linear-phase or minimum-phase impulse response table from a source table containing a frequency response or an impulse response.<br>
["exp"](../scoregens/genexp.md) - fills a table from an exponential formula.<br>
["farey"](../scoregens/genfarey.md) - fills a table from a Farey sequence.<br>
["padsynth"](../scoregens/genpadsynth.md) - fills a table using the padsynth algorithm.<br>
["quadbezier"](../scoregens/genquadbezier.md) - fills a table with a quadratic BÃ©zier curve.<br>
["sone"](../scoregens/gensone.md) - fills a table from a sone function formula.<br>
["tanh"](../scoregens/gentanh.md) - fills a table from a hyperbolic tangent formula.<br>
["wave"](../scoregens/genwave.md) - fills a table from a wavelet transform.<br>
