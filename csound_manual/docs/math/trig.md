# **Trigonometric Functions**

Opcodes that perform trigonometric functions are:

* [cos](../opcodes/cos.md), [cosh](../opcodes/cosh.md) and [cosinv](../opcodes/cosinv.md)
* [sin](../opcodes/sin.md), [sinh](../opcodes/sinh.md) and [sininv](../opcodes/sininv.md)
* [tan](../opcodes/tan.md), [tanh](../opcodes/tanh.md), [taninv](../opcodes/taninv.md), and [taninv2](../opcodes/taninv2.md)
* [signum](../opcodes/signum.md)
