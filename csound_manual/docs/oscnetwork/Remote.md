# **Remote Opcodes**

The Remote opcodes enable transmission of score or MIDI events through a network, so remote instances (or a different local instance) can process them. The following opcodes are available:


* [insglobal](../opcodes/insglobal.md) - Used to implement a remote orchestra.
* [insremot](../opcodes/insremot.md) - Used to implement a remote orchestra.
* [midiglobal](../opcodes/midglobal.md) - Used to implement a remote MIDI orchestra.
* [midiremot](../opcodes/midremot.md) - Used to implement a remote MIDI orchestra.
* [remoteport](../opcodes/remoteport.md) - Defines the port for use with the remote system.
