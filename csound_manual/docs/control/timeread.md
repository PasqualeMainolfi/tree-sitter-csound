# **Time Reading**

Opcodes one can use to read time values are:

* [elapsedcycles](../opcodes/elapsedcycles.md)
* [elapsedtime](../opcodes/elapsedtime.md)
* [eventcycles](../opcodes/eventcycles.md)
* [eventtime](../opcodes/eventtime.md)
* [readclock](../opcodes/readclock.md)
* [rtclock](../opcodes/rtclock.md)
* [timeinstk](../opcodes/timeinstk.md)
* [timeinsts](../opcodes/timeinsts.md)
* [times](../opcodes/times.md)
* [timek](../opcodes/timek.md)

You can obtain the system date using:

* [date](../opcodes/date.md) - Returns the number seconds since 1 January 1970.
* [dates](../opcodes/dates.md) - Returns as a string the date and time specified.

You can also set up counters using [clockoff](../opcodes/clockoff.md) and [clockon](../opcodes/clockon.md).
